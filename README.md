# ComPoseSearch
带文本框的搜索框
##Google Study Jam 学习校测验
## 一个用Commpose 编写的搜索框，虽然简陋，也算跨出第一步了吧

输入后会出现删除按钮，搜索控件有关键字回调，灭有填写管家字会有提示

点击搜索会记录下搜索历史在下方card，点击card item 可以键入数据，当然可以清空记录

代码很简单，就不怎么说明，相信你看得懂的！
